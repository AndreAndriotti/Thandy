>>> Como compilar o programa na plataforma disponibilizada

    Acesse a pasta Thandy/lab03.
    Para compilar o programa, utilizamos o gcc. 

        $ gcc lab03.c -o lab03

>>> Como executar o programa

    $ ./lab03

    Caso apareça um warning, apenas ignore.
